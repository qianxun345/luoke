# 洛克王国 - 向阳花全自动综合采集脚本

本项目包含 **两套** 基于 AutoHotkey (AHK) 编写的全自动脚本，旨在根据不同玩家的习惯，无人值守地刷取“向阳花”：
1. **`luoke_pgup.ahk` (图像识别版)**：完全依赖预设图片文件搜寻坐标及判定。
2. **`luoke_handle.ahk` (手动打点精准版)**：无视分辨率与搜图，靠玩家快捷键打点死记传送位置。

## 🌻 游戏核心自动化机制

两套脚本的核心都基于以下编排机制：
1. **自动调出地图/面板**：按下 `M` 键进入地图或其他交互面板（首次带有区域盲点）。
2. **判定与回场联动**：
   - **`luoke_pgup.ahk` (识图版)**：全屏搜寻 `image1.png`，找不到即滚轮上划重试；找到并在其中心偏下定点点击。通过识别 `image2.png` 判定场景加载。
   - **`luoke_handle.ahk` (打点版)**：彻底摆脱识图限制，直接单击用户事前用 `PgDn` 手动选取的传送位置，随后硬等 10 秒钟无脑度过加载黑屏期。
3. **动作刷新与重置**：回到主场景后，依次释放奇丽花，接着执行一套高频连招（Tab/2/Esc/R/X），重复 8 到 12 次拉满“产花”判定。

### 建议与要求
- 若使用 **打点版**，启动 `PgUp` 前 **必须** 用您的鼠标指着传送位置按下 `PgDn` 手动收录坐标，否则无法启动！

## 🛠️ 环境依赖与运行方式

> **提示**：仓库中提供的 `pgup.exe` 文件为已经打包好的独立可执行程序，如果您觉得方便且信任，可**直接运行 pgup.exe（建议右键“以管理员身份运行”）**，完全**跳过下方环境安装步骤**。

若您希望运行和编辑 `luoke_pgup.ahk` 源码脚本，需要先为电脑安装 AutoHotkey 运行环境：

1. **访问官网**：前往 AutoHotkey 官方网站 [https://www.autohotkey.com/](https://www.autohotkey.com/)
2. **下载安装包**：点击首页的 "Download" 按钮。环境版本需兼容 AHK v1.x 系列（如 v1.1.37.02）。
3. **安装环境**：双击下载的安装程序进行 "Express Installation" (快速安装)。
4. **运行代码**：直接对 `luoke_pgup.ahk` 文件**右键，选择“以管理员身份运行”**。

## 🚀 准备与使用说明

### ✨ 启动流程与热键 (`luoke_pgup.ahk` 识图版)
1. **屏幕适配**：
  - 方法一：游戏界面分辨率调整到1600*900，打开电脑设置，选择“系统” -> “屏幕”，将屏幕缩放改为125%，显示器分辨率为2560*1440。
  - 方法二：将屏幕缩放改为125%，自行截取image1.png和image2.png，下载并安装上述ahk，注意image1.png的左上侧与您传送点位置应保持110px个像素，具体查看像素可用微信截屏快捷键Alt + A查看。image2.png是传送到对应位置后，识别是否加载成功，作者选择用右上角地图来判断，截取时人物不要移动或转向，直接用微信快捷键截取。截取之后替换掉原有的文件。
2. **位置避让**：切勿将你的角色停留在 `image1.png` 要扫图的画面区域内，造成遮挡将导致无限向上滚动寻找。
3. **启动进程 / 开启按键**：打开脚本进程后，按下 **`PgUp`** 一键全托管刷花！

### ✨ 启动流程与热键 (`luoke_handle.ahk` 打点版)
1. **无需适配与截屏**：不管你啥分辨率、啥缩放率、也不需要避让角色站位，把鼠标移动到你要点进去的那个地图【传送点】的中心位置即可！
2. **录入坐标 (关键操作)**：在这个传送点上方，按下快捷键 **`PgDn`**，脚本上空会立刻弹出一行带有 `Recorded Target Point: x, y` 的提示框（只执行一次）。
3. **启动进程 / 开启按键**：当获取好上述传送黑屏点的坐标后，再按下快捷键 **`PgUp`** 即可大摇大摆开始硬莽挂机！

### 📜 连招底层原理（双版相同）
挂机一启动，就自动循环如下这套判定系统：
- **面板操作**：直接唤出 `M` 键。（首次入场会有固定坐标偏点击）
- **进图动作**：根据之前的方法（识图 / 录入硬点坐标），触发特定传送后，并随手盲点右下角交互框。
- **重新拉位**：识图版会自动卡帧去等 `image2.png`，打点版直接暴力等足 10 秒过图黑屏期！
- **连招爆发**：过图后，系统会自动用鼠标拉满场上第 1/3/4/5/6 号位的宠物互动。接着开启一套抽风连抽：`Tab切换` -> `2鞠躬判定` -> `Esc关页面` -> `R下坐骑` -> `X上坐骑`。（打满 8 到 12 个连环后停顿 1 秒重进大循环！）

## 🛑 工具通用关闭方法

如果您遇到异常无法停止进程的情况：
- 在桌面右下角任务栏系统托盘处找到带有 AHK 标志（默认绿色 `H`）的图标，**右键点击**并选择 **"Exit"** (退出) 即可彻底关闭当前运行的脚本。
