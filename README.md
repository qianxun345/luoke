# 洛克王国 - 向阳花自动采集脚本

本项目包含两个基于 AutoHotkey (AHK) 编写的自动化脚本，旨在利用《洛克王国》中的特定机制，帮助玩家稳定、高效地全自动刷取“向阳花”。

## 🌻 游戏核心机制简介

这两个脚本的运行基于以下游戏物理与交互机制：
1. **奇丽花产花**：游戏内特定精灵（奇丽花）拥有能够召唤或产出“向阳花”的机制。
2. **大范围采集**：玩家骑乘特殊精灵“迪莫”（拥有游戏内最大范围的自动采集判定）。
3. **动作刷新**：等待产花特效，如果未能产花，则立即重复特定动作（收回/放出 或 使用情感动作）来重置并刷新产出判定。
4. **游戏BUG**：若出现不产向阳花的情况，重新传送到刷花的对应位置即可。
通过不断循环触发动作，并利用迪莫骑乘状态下的被动大范围收集，玩家可以实现高效率的站桩农花。

### 建议地点
需要找一个没有野生精灵打扰的地方，比如学院驻地的魔力之源左边。

## 🛠️ 环境依赖与运行方式

> **提示**：仓库中提供的 `.exe` 文件为作者打包好的可执行程序，如果您觉得方便且信任作者编译的文件，可**直接运行 .exe 文件（建议右键“以管理员身份运行”）**，完全**跳过下方环境安装步骤**。

若您希望运行 `.ahk` 源码脚本，需要先为电脑安装 AutoHotkey 运行环境：

1. **访问官网**：前往 AutoHotkey 官方网站 [https://www.autohotkey.com/](https://www.autohotkey.com/)
2. **下载安装包**：点击首页的 "Download" 按钮。作者的 AutoHotkey 的版本是 v1.1.37.02。
3. **安装环境**：双击下载的安装程序（如 `AutoHotkey_Setup.exe`），选择 "Express Installation" (快速安装) 安装即毕。
4. **运行代码**：直接对相应的 `.ahk` 文件**右键，选择“以管理员身份运行”**。

## 🌻 一、 `luoke_home.ahk` （抱迪莫轮流扔花 - 全自动跑图收集）
此脚本是一套**全自动**长时间无人值守挂机方案。

### 📜 工作原理
要求将**一号位的迪莫设置为骑乘（一直抱着）状态**，以此最大化主角的收集范围。在这期间，脚本会自动轮流将背包中其他的奇丽花放出来尝试“扔花”（触发产花判定）并立刻收回，循环往复。

### ✨ 脚本特点
- **动作+跑图**：完美集成了动作高频判定和坐骑大范围收集的优势。
- **采集上限高**：虽然整体起花效率相对较低，但是可以拿六七十朵再传送一次来刷新地上的掉落数。

### 🚀 准备与使用
1. **备战**：登录游戏，移动到空旷场景。将队伍一号位放上**迪莫（保持随时骑乘坐骑的状态）**，其余位置放满**奇丽花**。
2. **启动**：在本目录下，**以管理员身份运行** `luoke_home.ahk` （或 `.exe` 版本）。看到系统右下角出现绿色 `H` 图标即为运行成功。
3. **开刷/停止**：切回游戏主画面，按下键盘上的 **`Home`** 键启动全自动挂机；当你想停止时，**再次按下** **`Home`** 键即可暂停脚本。
4. **⚠️ 注意事项（重要）**：脚本运行挂机期间，游戏画面内的**鼠标指针必须看向角色脚下的地面**！否则会导致放出的奇丽花离主角过远，奇丽花跑向玩家的过程会浪费时间，不仅效率低下且跑动期间**无法触发产花判定**！

## 🌸 二、 `luoke_pgdn.ahk` （全员奇丽花释放并鞠躬 - 高爆发）
此脚本是追求**瞬间爆发和最高产出效率**的方案。

### 📜 工作原理
脚本运行后，会自动控制将背包中的**奇丽花全部同时放出来**。接着，它将控制玩家带领这五只奇丽花同时执行**“鞠躬/礼貌”**情感动作；利用这瞬间产生的五个个体动作判定叠加，极高频地触发连环产花机制。

### ✨ 脚本特点
- **极其密集的产出**：同屏协同能够成倍增加判定机会，爆花概率与速度拉满。
- **采集上限提示**：当地面上掉落的向阳花数量到近 40 朵时，继续增加可能会导致部分花朵消失或卡顿无法获取；此时请务必（通过切图等方式）**刷新传送一次**来清空地图负荷。

### 🚀 准备与使用
1. **备战**：进入空旷场景。先确保**五只奇丽花**已经被全部释放并在场跟随；接着按下迪莫对应的数字键以**选中队伍里的迪莫（但切记不要上坐骑骑乘）**，建议放鞠躬动作快捷键的号位，有时候会。
2. **启动**：在本目录下，**以管理员身份运行** `luoke_pgdn.ahk`。右下角将出现绿色 `H` 图标即为运行成功。
3. **开刷/停止**：切回游戏主画面，按下键盘上的 **`PageDown` (PgDn)** 键即可开始爆发刷花；需要停止挂机时，**再次按下** **`PageDown`** 键即可暂停脚本。
